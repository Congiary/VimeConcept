О сайте

Статичный сайт созданный для работы c API VimeWorld, на сайте можно просматривать глобальную и месячную статистику, а также расширенную по ограниченному списку мини-Ходок. Такую как: средний показатель по Ходокам, сравнение показателей глобальной и месячной статистики. Благодаря rkkoszewski на сайте есть 3D визуализация скина и плаща Ходокока.
